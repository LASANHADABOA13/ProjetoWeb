O que é o Hardware ?

Hardware é um termo em ingles, que não tem uma tradução exata, mas se refere a toda parte fisica de um computador, mouse, teclado, monitor, gabinete, etc. Porem o hardware é dividido em duas partes, a parte externa e a parte interna

Parte externa

o hardware externo é como se fosse o "front end" (parte visual) dos computadores qualquer peça que é conectada diretamente no computador ou outro eletrônico e pode ser visto e tocado diretamente pelo usuário, são dispositivos como mouses, teclados, monitor, etc

Parte Interna

Dentro de seu computador existe os principais componentes para seu funcionamento e onde a magica acontece. Placa mae, processador (CPU) (e porfavor, CPU NÃO é gabinete!), placa de video, memoria ram e muitos outros componentes, o que ate dificulta o diagnostico de problemas (quando tem), por ter coisa ate demais dentro e nunca existe uma solução unica para todos os problemas
